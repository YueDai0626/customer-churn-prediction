# Customer Churn Prediction

## Project Overview
This project aims to predict customer churn using machine learning models.

## Tools & Technologies
- Python
- Pandas
- Scikit-learn
- Logistic Regression
- Random Forest

## Project Steps
1. Data cleaning and preprocessing
2. Feature engineering
3. Model training and evaluation
4. Business insights interpretation

## Results
The model achieved 85% accuracy on the test dataset and identified key churn drivers to support retention strategy.
